Welcome ようこそ

Generalities about Proximodo and how to configure your browser to use it.

Proximodo の概要とブラウザの設定。

Monitor 状況監視

How to use the Monitor screen. The monitor screen contains checkboxes to enable/disable general filtering, and statistics about current transfers.

状況監視画面の説明です。 フィルタの有効／無効を一括して切り替えるチェックボックスと、 現在の接続状況の統計が表示されます。

Settings 設定

How to use the Settings screen. The settings screen contains the proxy server parameters, filtering options, GUI options and lists.

設定画面の説明です。プロクシサーバの設定、 フィルタ動作の設定、GUI の設定などを行います。

Log Window ログウィンドウ

How to use to log window. It displays events generated by the filters or the proxy server.

ログウィンドウの説明です。ログウィンドウには、 フィルタやプロクシサーバで起きたイベントが表示されます。

Filters フィルタ

How to use the Filters edition window. This window allows you to modify existing filters. It is accessible from the Configuration screen.

フィルタ編集ウィンドウの説明です。 既存のフィルタを編集します。 フィルタ編集ウィンドウはフィルタ構成画面から開くことが出来ます。

Config 構成

How to use the Configuration screen. This screen allows you to organize filters, create new ones or activate/deactivate them in different configurations.

フィルタ構成画面の説明です。フィルタを新しく作成したり、 有効化／無効化して別のフィルタ構成を組むことが出来ます。

Syntax 文法

Summary of codes and commands you can use in match patterns and replacement texts.

マッチングパターンと置換テキストで使用できる、 メタキャラクタとコマンドの概要です。
