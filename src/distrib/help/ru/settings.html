Кнопки

Применить

Эта кнопка применит ваши настройки. Фактически, прокси-сервер продолжает работать с теми же настройками пока вы не нажмете эту кнопку (или соответствующий пункт меню). Если вы изменяете или минимизируйте окно, Proximodo спросит вас примененить ли новые настройки. Если вы сохраняете настройки используя меню, ваши изменения также вступят в силу.

Отменить

Нажав на эту кнопку вы отмените все изменения сделанные вами в этом окне. Используйте ее если вы думаете, что совершили ошибку (например стерли имя списка или адрес прокси) и хотитесь вернуть настройки назад в их первоначальное значение, когда вы открыли это окно.

Настройки фильтрации

Bypass URL Pattern

Это поле содержит образец для выбора страниц, которые не должны фильтроваться. Если URL (без ://) документа/изображения совпадает с этим образцом, он не фильтруется в текущей конфигурации. Тем временем, все другие страницы будут отфильтрованы в соответствии с настройками. По умолчанию $URL(Bypass) означает, что каждый URL в списке "Bypass" будет пропускаться без фильтрации. Это - более удобно чем перечислять страницы в этом небольшом поле! Если есть страницы или сайты которые вы не хотите фильтровать, добавьте строку в список "Bypass" вместо отключения фильтров Proximodo.

Интерфейс пользователя

Язык

Это поле позволяет вам выбрать какой язык интерфейса будет использоваться при работе программы. Изменения вступят в силу после перезагрузки Proximodo.

Показывать GUI при запуске

По умолчанию, окно Proximodo открывается каждый раз при запуске. Если вы хотите, чтобы окно не открывалось, а программа сворачивалась в системный трэй снимите флажок с этого переключателя. Для последующего открытия окна нажмите на иконке в трэе.

Запустить при запуске Proximodo

По умолчанию, при запуске Proximodo запускается ваш браузер. Вы можете изменить такое поведение, сняв флажок. Или измененить запускаемое приложение (например firewall). Используйте кнопку для поиска приложения на диске.

Настройки прокси

Порт

Это порт который использует Proximodo. Выберите любое значение между 1024 и 65535 (кроме 8080 или 8088, т.к. это стандартные величины), затем установите такое же значение в настройках браузера вместе с "localhost" в поле адрес (или IP вашего компьютера для удаленного соединения с Proximodo). Вы должны настроить свой firewall для правильной работы Proximodo.

Разрешить URL команды

Когда вы включите эту опцию, вы можете заставить Proximodo дополнительно обрабатывать страницы с определенным префиксом - "URL команда". URL команды:

  • bin.. : пропускает страницу через фильтр входящих заголовков без изменения
  • bout.. : пропускает страницу через фильтр исходящих заголовков без изменения
  • bweb.. : пропускает страницу через фильтр текста без изменения
  • bypass..: пропускает страницу через все фильтры без изменения
  • dbug.. : вставляет отладочные маркеры там где сработал фильтр
  • src.. : показывает код страницы

Префикс

Для того, чтобы запретить сайтам использовать URL команды для блокирования фильтров на их страницах, вы можете модифицировать URL команды использовав дополнительный префикс. Например, с префиксом foo URL команда bypass.. станет foobypass..

Использовать удаленный прокси

Когда этот переключатель установлен, Proximodo работает через указанный прокси-сервер. В противном случае Proximodo будет работать напрямую.

Прокси

Если вы хотите, или это необходимо, иметь доступ к интернету через другой прокси-сервер, впишите значение адрес:порт своего прокси-сервера в этой области. Так же вы должно сделать это если ваш компьютер подключен к интернету через локальную сеть. Вы можете выбрать ранее использовавшийся прокси из выпадающего списка. Этот список также используется вашими фильтрами, содержащими команду $USEPROXY. Для того, чтобы добавить прокси к списку, просто впишите его адрес:порт и нажмите [Enter]. Чтобы удалить прокси из списка, выберите его в списке и нажмите кнопку справа. Если вы впервые настраиваете Proximodo, возможно вы захотите использовать значение прокси-сервера установленные сейчас в вашем браузере, тогда используйте localhost:8080.

Разрешить IP диапазон

Если этот переключатель выключен, только программа на локальном компьютере может подключиться к Proximodo. Если он включен, то соединения от отдаленных компьютеров могут быть приняты (если их IP находится в пределах установленного диапазона). Используйте с осторожностью, только если вы действительно хотите предоставить доступ к интернету другим пользователям вашей локальной сети (например детям).

Начало и конец диапазона

Если переключатель "Разрешить IP диапазон" включен, то эти два значения устанавливают диапазон IP адресов с которых можно соединяться с Proximodo. Делайте этот диапазон как можно уже в соответствии с вашей локальной сетью.

Списки

Имя

Списки это файлы загруженные в память и чьи строки являются шаблонами, которые могут использоваться один после другого фильтрами с командой $LST. Это - очень мощная функция позволяющая разделить шаблоны между фильтрами, и сделать их динамическими: вы можете добавить новые шаблоны вручную или автоматически в список, и они немедленно будут обрабатываться фильтрами, которые их используют. Большой плюс, что списки могут содержать сотни строк, а фильтры остаются небольшими и их не надо изменять.
- Чтобы добавить новый список напишите имя и нажмите кнопку Плюс, затем выберите его из выпадающего списка и выберите путь к файлу.
- Чтобы переименовывать список, выберите его измените имя и нажмите [Enter].
- Чтобы удалить список, выберите его и сотрите его имя и нажмите [Enter].
- Чтобы просмотреть / изменить содержимое списка, выбираться его и нажмите кнопку с изображением лупы справа.

Файл

Это путь к файлу выбранному из списка. Вы можете вписать сюда путь к файлу или выбрать его нажав кнопку справа. Если файл не существует, Proximodo спросит нужно ли создать файл.
